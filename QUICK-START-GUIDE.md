## ğŸ“š Using FlashQ - Simple Steps

### For Parents: Creating Flashcards

1. **Launch FlashQ** (double-click desktop icon)
2. **Click "Create Flashcards" tab**
3. **Select a Subject** (Math, Science, etc.) or create a new one
4. **Fill in the form:**
   - Main Topic: What you're studying (e.g., "Multiplication Tables")
   - Sub Topic: More specific (e.g., "Times 5")
   - Number of Flashcards: How many you want (5-20 recommended)
5. **Upload photos** of your child's notes or textbook pages
6. **Click "Generate Flashcards"**
7. Wait while FlashQ:
   - Reads the text from photos (OCR)
   - Uses AI to create flashcards
8. **Review and edit** the flashcards if needed
9. **Click "Save Flashcard Set"**
10. Done! Ready to study!
